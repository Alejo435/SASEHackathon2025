## SCHack - Sunset

  Sunset is a social app created for the STEM Connect Hackathon Fall 2025
 ## Setup Instructions

1. Clone the Repository
```bash
git clone https://github.com/your-username/sunset.git
cd sunset
```
2. Install Dependencies
```bash
pip install -r requirements.txt
```
3. Run the Program
```bash
python3 app.py
```
